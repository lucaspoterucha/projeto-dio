HTML — Linguagem de Marcação

O HTML é a base de toda a web. Ele é responsável por estruturar o conteúdo dos sites que acessamos todos os dias.

O que é HTML?

HTML significa Linguagem de Marcação de Hipertexto. Ele utiliza tags para organizar textos, imagens, links e outros elementos em uma página web.

Diferente de linguagens de programação, o HTML não possui lógica. Ele serve para estruturar o conteúdo.

Um pouco da história

O HTML foi criado por Tim Berners-Lee em 1991. Desde então, passou por várias versões. Algumas tags antigas, como font, hoje são consideradas obsoletas.

Mesmo assim, elas ainda ajudam no aprendizado inicial. Hoje em dia não usamos mais essas tags em projetos profissionais.

Principais Tags do HTML

Títulos

O HTML possui seis níveis de títulos, do h1 ao h6, usados para hierarquia.

Listas

Existem dois tipos principais de listas:

  • Lista não ordenada (ul)
  • Itens da lista (li)
  1. Lista ordenada (ol)
  2. Sequência numérica

Formatação de Texto

Podemos deixar o texto em negrito, itálico, sublinhado ou destacado.

Também é possível usar textos menores, subscrito como H2O ou sobrescrito como 102.
